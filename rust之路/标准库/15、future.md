# std::future

异步值

## structs

| Pending | 实验阶段：创造了一个永远不会解决的future，代表着一个永远不会结束的计算。 |
| ------- | ------------------------------------------------------------ |
| Ready   | 试验阶段：创造一个立即具备价值的future。                     |

## Traits

| Future     | 异步计算               |
| ---------- | ---------------------- |
| IntoFuture | 实验阶段：转换为future |

## Functions

| pending | 实验阶段：创造了一个永远不会解决的future，代表着一个永远不会结束的计算 |
| ------- | ------------------------------------------------------------ |
| ready   | 试验阶段：创建一个立即准备好的future                         |
